# Guide Image Prompts — Realistic / Lifestyle Photography

Replacement images for the **Cost Guides** section. 21 article images + 5 category headers.
Style: warm, photorealistic lifestyle photography (no illustration, no mascots).

## How to use these
1. Generate each at **1024×1024 (square)**, PNG. The cards crop to roughly square.
2. The **STYLE RECIPE** is repeated in every prompt on purpose — it keeps all 21 photos looking like one cohesive set. Don't change it.
3. For best consistency, generate them in **one ChatGPT session**; after the first you can say *"same exact style and lighting as the previous image"* before pasting the next scene.
4. **No text in the images** — the cards add their own titles, and AI text renders badly. Coins are fine; avoid banknotes (region-specific + texty).
5. Save each file using the **filename** noted under each prompt. Suggested location: `/public/guides/`.
6. Tell me when they're generated and I'll wire the new filenames into `app/guides/page.tsx` (the `guideImages` map).
